# 检索流程

## 1. 确定输入和章节

用户可能提供图片路径、粘贴图片、或文字描述荷载。章节按用户说的来，不确定就问。

## 2. 执行检索

**有图片** → 直接跑脚本，不要先用视觉工具看图：
```bash
python "D:\桌面\答疑、帮做\结构力学\帮做\search.py" search --image "<图片绝对路径>" --chapter "<章节>"
```

**只有文字描述** → 构造荷载 JSON：
```bash
python "D:\桌面\答疑、帮做\结构力学\帮做\search.py" search --loads '{"loads":[{"type":"集中","raw":"10kN"}]}' --chapter "<章节>"
```

荷载 JSON 规则：type 只能是 集中/均布/弯矩，raw 保留原标注，按 集中→均布→弯矩 排序。

## 3. 输出结果

直接展示脚本输出的 Top 5 结果即可。脚本已自动打开全部匹配图片。

**禁止**：不要用 MCP 视觉工具再去看结果图片做二次对比。相似度算法已经做好了匹配，信任它。
